# Python Data Structures

Create the well known Data Structures in Python.

  - Linked List
  - Stack
  - Queue
  - Binary Search Tree (BST)

### Usage

```sh
$ git clone https://github.com/kakatics/pythonDS.git
$ cd pythonDSA
$ cd stack
$ python3 -i Stack.py # Load the DS in the Python Emulator
```
